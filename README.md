### 💼 Where I've Worked

**Research Associate @ UB** — Building WiFi-based indoor localization systems using deep learning with federated learning and differential privacy. Training CNN models on AoA-ToF heatmaps, achieving sub-meter prediction accuracy. Discovered that 2D DFT beamforming provides ~5.5× natural noise suppression, making input-level differential privacy practical.

**ML Engineering Intern @ New Era Cap** — Built an end-to-end demand forecasting pipeline for global supply chain planning using Azure ML, Snowflake, and ensemble models (XGBoost/LightGBM). Automated data flows, model retraining via MLflow, and built a planner-facing interface for forecast review and validation.

**Software Development Engineer @ AppViewX** *(~1.5 years)* — Independently proposed and built a multi-agent RAG onboarding chatbot using LangGraph and OpenAI API — POC delivered in 3 weeks, recognized by the VP of Engineering. Optimized MongoDB queries from 5 min → 300–400ms across 150M+ records. Built a Java-based CA simulator saving ~$700K/year. Developed production REST/SOAP APIs and microservices with Spring Boot, Kafka, and Flink.

**@ UB Research (Conversational AI)** — Fine-tuned LLMs for personalized assistive communication, designed a RAG architecture with vector databases, and deployed a real-time voice-enabled agent. Achieved BERTScore F1 of 0.84.

**@ Honeywell Aerospace** — Engineered a domain-specific text-to-speech system for Air Traffic Control communications, improving pronunciation accuracy and clarity in cockpit communication.
